 # Circulating miRNA Profiles in Stroke Patients vs Healthy Controls

## Project Purpose :

**What classes exist** 
This project analyzes **small RNA sequencing (miRNA-Seq)** data from peripheral blood plasma to investigate **differentially expressed microRNAs (miRNAs)** associated with **ischemic stroke**.  

The dataset compares miRNA expression between **stroke patients** and **healthy individuals**, aiming to identify potential **circulating miRNA biomarkers** for stroke diagnosis, prognosis, or molecular mechanisms of cerebrovascular injury.  

**What classes exist**
Two main classes exist in the dataset:  
- **stroke patient** samples  
- **healthy** (control) samples  

**Why dataset is useful**
This dataset is useful because circulating miRNAs in plasma are promising non-invasive biomarkers for stroke. They reflect disease-specific changes in gene regulation, may distinguish stroke from other conditions, and offer insights into post-stroke biological processes — all from an easily accessible peripheral blood source.
